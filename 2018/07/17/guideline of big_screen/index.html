写在前面的话

门卡一定要记得带出来！！！

没事不要去关闭任何大屏服务器！！！

用完了关闭 VNC Viewer 就行，不要关机！！！

也不要没事随意去动服务器的IP设置，恢复起来真的很麻烦！！！

更不要随意去禁用本地连接，就算你禁用重启一般也无法改变上不了网的事实，只会让后续恢复操作更繁琐！！！




大屏使用步骤

1. 首先你需要拿到大屏的门卡，大屏是在410-2，进去后，右手边是灯，左手边是空调，进去后先把门卡揣兜里，千万别把门卡放桌上，容易忘记带出来!

2. 桌上有两个大屏的遥控器，都是通用的，对着大屏按下“ON”键，打开大屏。注：第一次按下“ON”可能不能开启所有的屏幕（我也不知道为啥），这时候就要多按几下。

3. 屏幕开启后，使用桌上的华硕笔记本，打开桌面上的 MoController2.4.6.9（一个黑色小方块的图表），在预案管理里，选择 WEB05（ WEB05 是方形像素适配，大屏的两边会有黑边） 或者 B05（ B05 是通过像素拉伸使得全屏铺满），一般为了视觉效果，使用的都是 B05。选择好之后，大屏上应该已经能显示所选方案的服务器的画面了。

4. 选择好适配方案后，在开始菜单中搜索 VNC Viewer ，打开，对应输入入如下:

VNC Server：192.168.10.20x (最后八位ip根据你选择的预案切换，如果选了B05，就是205，如果是B01，那就是201)

Encryption: Let VNC Server choose

之后点击 connect，成功连接后输入如下：

Username： administrator

password： !Qaz2wsx

5. 现在你已经连上大屏服务器了，可以使用它了。 注：在大屏服务器里，不要按”enter“键，因为他和 MoController2.4.6.9 的方案切换热键冲突，按了会直接切换方案。




可能遇到的问题

问题一：找不到预案管理

原因：MoController2.4.6.9 没连接上大屏

解决方法：打开MoController2.4.6.9后，左上角“ 软件操作 ” -> “ 连接 ”，就会看到下面有预案管理。


问题二：MoController2.4.6.9 无法连接

原因：笔记本的wifi没连上或者没连对

解决方法：将笔记本的wifi连上“ HUAWEI-410 ”，然后重试。


问题三：大屏服务器 （B05 或 WEB05 或其他） 无法上网

原因1：B01 的 cow 没开，无法给其他服务器提供代理服务

解决方法：用 VNC Viewer 以同样的方法（相同的账号密码）连上IP我为： 192.168.10.201 的服务器，上去之后，在桌面有个名为“ cow-win64-0.9.8 ”的文件夹，在里有个 cow 的可执行程序，直接运行即可。

原因2： B01 无法上网

解决方法：一般遇到这种问题，起因就可能比较多面了，我们需要逐层排查：

在此之前，再强调一遍，绝对不要禁用本地连接，一旦禁用就要去411机房直接对服务器操作，很麻烦！！！

1.首先可以关闭一下cow，看一下能不能上网，如果不行，则打开网络连接设置

2.B01的本地连接有两个，第一个是用于服务器之间的连接的，第二个是用于连接外网的，一般上不了网是第二个连接出问题了，可能是第二个连接的ip冲突了，需要联系李伟清老师，或者王召喜老师解决

3.服务器连接路由器的网线出问题了，需要去机房检查一下路由器是否工作正常，网线连接是否正常。


问题四：VNC Viewer 连不上大屏服务器（ B01，B05， WEB05 等）

原因1：服务器没开机或死机

解决方法：使用开关机卡的终端发送开机指令，具体操作：

1.打开文件夹 “ 开关机调测工具及指令说明 ”里面的 ” 设置程序 “

2.在服务器列表中，选中你要进行操作的服务器，点击 ” 读取/修改设备参数 “，就可以看到该服务器的ip等信息。

3.打开同一文件夹中的 ” 网络测试助手 “，输入如下：

协议类型：TCP客户端

本地ip地址：192.168.10.8x （此处填入你之前用“设置程序”查看的对应服务器的ip）

本地端口号：3000 （此处填入你之前用“设置程序”查看的对应服务器的端口号）

点击 “ 连接”

4.链接成功后，在下方输入框中输入状态读取指令read，并点击” 发送 “。

5.如果返回值为： stopped，说明没开机，此时输入“ start ”就可以进行开机；如果返回值为：running，说明之前服务器死机了，此时需要强制关机，输入“ forced
”，当返回值为stopped时，说明成功关闭服务器，之后就可以输入“ start ”进行开机。（具体及其他指令内容详见同一文件夹下的 ” 开关机卡操作手册.docx “）

刚开机的时候，要耐心等待一会儿，大约一分钟之后，当开机界面运行到输入用户名和密码时，就可以用 VNC Viewer连接了，这个时候 VNC Viewer的连接 ，要多试几次，一般两三次之后就可以连上服务器，开机的密码和 VNC Viewer 的连接密码相同。

原因2：服务器IP设置出错

解决方法：使用设置程序远程设置，具体操作：

1.打开文件夹 “ 开关机调测工具及指令说明 ”里面的 ” 设置程序 “

2.在服务器列表中，选中你要进行操作的服务器，点击 ” 读取/修改设备参数 “，就可以看到该服务器的ip等信息，同时，也可以更改相关参数（一般我们需要改的参数就IP地址和子网掩码，其余的可参照其他服务器），正确的参数如下：

IP：192.168.10.8x (最后八位ip根据你选择的服务器进行调整，如果选了B05，就是85，如果是B01，那就是81)

子网掩码：255.255.255.0

原因3：服务器网络被禁用

解决方法：很多人在发现某服务器上不了网之后，会下意识地去禁用本地连接，再重新启用，殊不知，一旦禁用了本地连接，就不能直接在大屏房间里操作了（gg），需要直接去411找老师开机房门，将显示器、键盘、鼠标直接连上服务器，直接对服务器进行网络设置，需要说明的是，服务器的vga端口好像不太好用，我试了好多次，好多台服务器，都连不上显示器，这里推荐一个笨办法， 那就是两个人微信开个视频聊天，一个人站在大屏房间，一个在机房操作服务器（蠢，但有效）。


问题五：大屏遥控器按了半天还是没能打开大屏（或者只打开了几块屏幕）

原因：大屏硬件损坏（例如：大屏的红外线接收器坏了，导致没法遥控开启）

解决方法：找411管大屏的老师，让他帮忙维修。


问题六：门卡落里面了

原因：自己面壁思过去！

解决方法：去陈为老师组借他们那儿大屏的门卡。
